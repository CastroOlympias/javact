# Javact

**Java + React** | Spring Boot GraphQL BFF + React 19 Client

Coined **Javact** to represent the decoupled Spring Boot Backend-for-Frontend (BFF) architecture paired with a high-velocity Vite + React + Apollo Client frontend.

### 🏗️ C4 Architecture Paradigm
Derived from custom full-stack system patterns:
* **Collections:** Domain entity models & persistent data layer
* **Connections:** Spring Boot GraphQL resolvers & API gateways
* **Contracts:** Schema definitions, DTOs, and Apollo Client queries
* **Configurations:** Local-first state management, security guards, and Vite proxy rules
